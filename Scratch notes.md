
# **Harmony Ideas**

Traditional approach:

```text
Melody
↓
Add Harmony
```

Possible alternative exercises:

### **Harmony First**

Given harmony.

Student writes melody.

### **Missing Voice**

Given outer voices.

Student fills in middle voice.

### **Missing Melody**

Given harmonic progression.

Student creates melody using known motive types.

### **Motive Recognition**

Student identifies:

- Motives
- Variations
- Development techniques

inside existing pieces.

---

# **Project-Based Learning**

Instead of:

```text
Lesson
↓
Exercise
↓
Quiz
```

Use:

```text
Project
↓
Need a concept
↓
Learn concept
↓
Apply concept immediately
```

Example:

Project:

Write a Four-Bar Melody

Image:

Lonely traveller at dusk.

Gesture:

Hesitation.

Available motive types:

- Stepwise
- Repeated note
- Leap

Development tools:

- Repetition
- Sequence

Student creates a melody immediately.

Theory is introduced only when needed.

---

# **Repetition & Mastery**

A coding-style approach may work well.

Instead of:

Project 1 → never use skill again

Use:

Project 1

- Motive
- Melody

Project 2

- Same skills
- Different image

Project 3

- Same skills
- One new concept

Project 4

- Same skills
- New emotional context

The same tools are repeatedly reused until they become natural.
